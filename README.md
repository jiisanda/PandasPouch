# 🐼 pandas-pouch 🐼

A Distributed Caching Service with Rust 🦀🦀.


## Progress

- [x] Basic Cache
- [ ] LRU Cache
  - [x] LRU Cache with TTL
  - [x] LRU Cache with Size
  - [x] LRU Cache with Size and TTL
  - [ ] print cache
  - [ ] Thread Safety


## Usage

Run examples as follows:

```bash
source .bashrc
```

```bash
pandas-pouch --example <file-name-from-examples>
```
